# Theory status

## 1. Starting hypothesis

The original working hypothesis was:

> Particles may be representable as stable resonance or concentration patterns in an underlying field.

Initial language was intentionally broad: atoms, electrons and elementary particles were discussed as oscillatory or resonance-like structures. The work then moved from conceptual speculation into explicit field models.

## 2. Scalar field phase

The first simulations used scalar nonlinear Schrödinger / Gross-Pitaevskii-type dynamics. These models produced stable vortex-like defects under conservative split-step Fourier evolution.

Key finding:

- scalar vortices can be stable and trackable;
- they do not naturally produce clean full braiding in the tested regimes;
- additional phase forcing often creates unwanted defects.

Conclusion:

> scalar vortex dynamics alone is not a sufficient model for robust particle-like topological sectors.

## 3. Two-component chiral spinor phase

The model was extended to a two-component field

\[
\Psi(x,y,t)=
\begin{pmatrix}
\psi_a\\
\psi_b
\end{pmatrix}
\]

with chiral/Dirac-like derivative coupling. This introduced an internal Pseudospin

\[
\mathbf{s}
=
\frac{\Psi^\dagger \boldsymbol{\sigma}\Psi}
{\Psi^\dagger\Psi}.
\]

This phase produced:

- stable two-defect configurations;
- strong internal spin texture dynamics;
- non-trivial global phase / holonomy;
- weak external orbiting motion;
- no robust quantized Skyrmion charge.

Important interpretation shift:

> the physically relevant structure is not a point-like vortex core, but an extended chiral spin texture.

## 4. Why quantization initially failed

The chiral spinor model did not impose enough topological structure. In particular, it lacked:

- an explicitly compact target space;
- fixed boundary conditions;
- a clean map R² -> S²;
- a topological sector classification.

Therefore the Skyrmion-like charge drifted continuously.

## 5. CP¹ / Sigma model phase

The successful reformulation uses a normalized spin field s(x,y) in S² with fixed north-pole boundary conditions.

The CP¹ / nonlinear sigma model with approximate Skyrme stabilization produced robust quantized sectors:

Q ≈ ±1, ±2, ±3.

This is the current theoretical core.
